> 아래 항목들은 트래픽이 월 1,000 세션을 넘긴 뒤에 구현해야 ROI가 나는 항목들입니다.
> 지금 하면 시간 낭비. 트래픽 지표 확인 후 순서대로 진행할 것.

## 1. 리드 판매 파이프라인 구축

### 왜 나중인가
리드를 받을 시공업체가 없으면 리드 폼은 빈 깡통.
트래픽이 100/월이면 리드가 월 1~2건인데, 이걸로 업체를 설득할 수 없음.

### 구현 내용
- `sellers.json` — 주/카운티별 시공업체 커버리지 맵
- 3-Tier Fallback 라우팅 (직접→네트워크→제휴)
  - Tier 1: 계약 업체 → 직접 리드 전달 (리드당 $50~$150)
  - Tier 2: Networx, Modernize 같은 리드 어그리게이터 API 연동 (리드당 $10~$30)
  - Tier 3: 아마존 제휴로 테스트킷/모니터 추천 (클릭당 $1~$5)
- `LeadController.java`에 fallback 로직 추가
- `logs/lead_deliveries.csv` — 수익 귀속 추적

### 시작 조건
- 월 트래픽 1,000+ AND 리드 폼 제출 20+/월

## 3. 상업/다세대 B2B 리드 채널

### 왜 나중인가
B2B 리드는 단가가 높지만($100~$500/건), 
상업 시공 업체를 직접 영업해야 하므로 영업 시간이 필요.
주거 리드가 월 50건 이상 나오는 안정 궤도에 올라간 뒤에 확장.

### 구현 내용
- 카운티 페이지에 CTA: "Need a commercial or multi-family estimate?"
- 별도 리드 폼: 건물 유형, 면적, 유닛 수, 연락처
- 별도 CSV: `logs/commercial_leads.csv`
- 새 페이지: `pages/commercial_lead.jte`
- 새 컨트롤러: `GET /commercial-estimate`

### 시작 조건
- 월 주거 리드 50+건

## 4. 팬 교체 / 유지보수 pSEO

### 왜 나중인가
기존 저감 시스템 설치 가정(2000~2010년대)의 팬이 수명 다해서 교체 필요한
수요가 있지만, 이건 완전히 새로운 키워드 클러스터.
메인 키워드("mitigation cost")로 오서리티를 먼저 잡고 나서 확장해야 함.

### 타겟 키워드
- "radon fan replacement cost"
- "radon manometer reading zero"
- "radon fan not running"
- "RadonAway RP145 replacement"

### 구현 내용
- `pricing_config.json`에 `fan_replacement` 섹션 추가
- `/radon-fan-replacement/{state}/{county}` — 새 pSEO 페이지 계열
- 아마존 제휴: 교체용 팬(RadonAway RP145/RP265, Fantech Rn2/Rn3) 직접 링크

### 시작 조건
- 메인 키워드("radon mitigation cost [county]")로 월 3,000+ 트래픽 확보 후

## 5. CTA "3곳 무료 견적 비교" 강화

### 왜 나중인가
업체 네트워크가 없으면 "3곳 비교"라는 약속을 지킬 수 없음.
빈 약속은 신뢰를 깎음. 업체 1~2곳이라도 계약된 후에 표시.

### 구현 내용
- 영수증 하단에 "Get 3 Free Quotes from Certified Pros" 버튼
- 클릭 시 리드 폼으로 스크롤 또는 모달
- 폼 제출 후 매칭 결과 페이지 표시

### 시작 조건
- 최소 2개 주에서 시공업체 계약 완료

## 6. 가격 정확도 보강 — 실제 시공 데이터 수집

### 왜 나중인가
현재 가격은 국가 평균 + 지역 멀티플라이어 추정치.
실제 시공 가격 데이터를 모으려면 사용자 제보 또는 업체 협력이 필요하고,
모두 트래픽이 전제조건.

### 구현 내용
- "실제로 시공하신 분이라면 가격 공유해주세요" 폼 추가
- 수집된 데이터로 카운티별 실제 평균가 계산
- 이게 축적되면 독점 데이터(MOAT)가 됨 → 경쟁자 복제 불가

### 시작 조건
- 월 트래픽 5,000+ (데이터 제보가 통계적으로 유의미해지는 시점)
